## 정리 방식

- 책 성격에 따라 챕터 또는 아이템 단위 `.md` 파일로 분리
- 전문가 수준으로 작성 (기초 설명 생략)
- 한국어 기술, 전문 용어는 영어 병기 예: 캐시 라인(cache line)
- ASCII 다이어그램으로 구조 시각화
- 핵심 코드 예제 포함
